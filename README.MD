## 01 서비스 소개

> #### 주짓수 대회 자동화 서비스
> 
> "Royale"은 아날로그로 진행되면서 발생하는 불편함들을 자동화한 서비스이다.

---

## 02 주요 기능

**❗ 스태프**

&nbsp;&nbsp;✔ 대회 생성

&nbsp;&nbsp;✔ 대회 참가 마감

&nbsp;&nbsp;✔ 대진표 생성

&nbsp;&nbsp;✔ 스코어보드 관리

&nbsp;&nbsp;✔ 선수 계체 관리

<br/>

**❗ 선수**

&nbsp;&nbsp;✔ 대회 참가 신청

&nbsp;&nbsp;✔ 스코어보드 조회

&nbsp;&nbsp;✔ 대회 현황 조회
